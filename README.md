# Ruhama Glass Web

Marketing site for Ruhama Glass showcasing services, projects, and contact info.

## Getting started

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Notes

- Brand assets (logo and photos) live in `public/`.
- Typography uses Cormorant Garamond (display) and Manrope (body) via Google Fonts.
